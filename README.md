# 💙 Mumbai Indians Fan Website

A modern, interactive, and responsive fan website dedicated to the Mumbai Indians (MI), one of the most successful franchises in the Indian Premier League (IPL).

This website serves as a one-stop destination for MI fans to stay updated with team information, player profiles, match schedules, statistics, latest news, and fan engagement activities.

---

## 🌟 Features

### 🏏 Team Overview
- Introduction to Mumbai Indians
- Team history and achievements
- Franchise information
- Team branding and colors

### 👥 Player Profiles
- Current squad details
- Player roles and positions
- Player statistics
- Individual player cards

### 📅 Match Schedule
- Upcoming fixtures
- Match dates and timings
- Opponent details
- Venue information

### 📊 Team Statistics
- IPL titles won
- Matches played
- Win percentage
- Team records and achievements

### 📰 Latest News
- Recent team updates
- Match reports
- Transfer and squad news
- Tournament updates

### 🎉 Fan Zone
- Interactive fan engagement section
- MI chants and slogans
- Fan messages
- Community support section

### 📱 Responsive Design
- Mobile-friendly interface
- Tablet compatibility
- Desktop optimized layout
- Smooth animations and transitions

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/DivyanshShukla001/Mumbai-Indians.git
```

### Navigate to Project Folder

```bash
cd Mumbai-Indians
```

### Open in Browser

Simply open:

```bash
index.html
```

Or use Live Server in VS Code.

---

## 🔥 How to Run Locally

### Using VS Code Live Server

1. Open project folder in VS Code.
2. Install **Live Server Extension**.
3. Right-click on `index.html`.
4. Click **Open with Live Server**.

The website will launch automatically in your browser.
